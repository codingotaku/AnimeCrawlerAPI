# AnimeCrawlerAPI
A java based Anime crawler API with multiple server support. 

Currently the API is under development 

For Maven, use

```xml 
<dependency>
	<groupId>com.codingotaku.api</groupId>
	<artifactId>anime-crawler-api</artifactId>
	<version>0.1</version>
</dependency>
```
